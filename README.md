## Description
You have left from your own dimension for greener pastures. In The Winston Game, travel to multiple dimensions in a completely aimless quest. 
As time goes on, you might just make the right pathway through a world to help someone. You are in a time loop until you figure out the correct route through each realm, and you unlock the ability to go to the final realm.

The files in this project include the Read Me file -- the one you're reading right now, The Project Proposal, describing the basic features of the game and the plan to create it, the requirements file, which lists the libraries that are needed, and the main game file, which has the text adventure game that will be played. An area of improvement would be having more visual elements. I had some, but more would be better. Using Pygame to make a window would have added a lot
